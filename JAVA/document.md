# PKCE - Java Implementation

## Prerequisite
- Install JDK 11 or newer

## Usage
1. Navigate to the project directory:

```bash
cd pkce-java
```

2. Compile:

```
javac Main.java lib/PKCE.java
```

3. Run

```
java Main
```

4. Sample Output:

```
code_verifier: <random_value>
code_challenge: <hashed_value>
code_challenge_method: S256

```
