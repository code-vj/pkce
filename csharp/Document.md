# PKCE - C# Implementation

## Prerequisite
- Install [.NET SDK 6.0+](https://dotnet.microsoft.com/download)

## Usage
1. Navigate to the project directory:
   ```bash
   cd pkce-csharp
```

2. Compile and run the sample application:

```
dotnet new console --force
dotnet build
dotnet run
```
