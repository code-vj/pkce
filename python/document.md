# PKCE - Python Implementation

## Prerequisite
Install Python 3.x

## Usage
1. Navigate to the project directory:

```bash
cd pkce-python
```

2. Run the sample app:

```
python main.py
```

3. Output

```
code_verifier: <random_value>
code_challenge: <hashed_value>
code_challenge_method: S256
```
